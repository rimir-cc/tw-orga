# orga

> Organisational domain model built on the typed plugin

Provides type definitions for common organisational entities like projects, tasks, teams, persons, milestones, and roadmaps.

## Key features

* **Pre-built type definitions** -- project, task, team, person, workstream, assignment, roadmap, milestone, 1on1
* **Hierarchical node structure** -- parent relationships between organisational entities
* **Field definitions** -- common attributes like status, dates, stakeholders, participants
* **Enum definitions** -- predefined value sets (e.g., core-status)
* **Extensible** -- builds on the typed plugin's inheritance model

## Prerequisites

* typed plugin must be installed

## Quick start

Install plugin alongside typed. Use the Spawner to create new organisational entities. All types are immediately available in the type system.
